Healthcheck command
Interval

Duration to initially wait before executing the health check and then the frequency at which subsequent health checks will be performed.

Timeout

Duration that Docker awaits for your health check command to return an exit code before declaring it as faimed.

Start period

Duration your container needs to bootstrap. During this period, health checks with an exit code greater than zero won't mark the container as unhealthy.

Retries

Number of consecutive health check failures required to declare the container as unhealthy.
